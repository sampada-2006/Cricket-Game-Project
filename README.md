#Cricket Game – Bat Ball Stump
About the Project
-This is a simple and fun Cricket Game built using HTML, CSS, and JavaScript.
-The game is played between: User and Computer
-Both players choose one of the three options: Bat , Ball or Stump

How to Play
-The user selects one option and the computer randomly selects one option.
-The result is displayed:User won or Computer won or It's a tie

Game Rules
-Bat beats Ball
-Ball beats Stump
-Stump beats Bat
